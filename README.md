## Running the project locally
* Make sure your [NodeJS](https://nodejs.org/en/) and yarn installed.

* Install your dependencies
```bash
$ cd path/to/rpg-api/
$ yarn install
```

* Generate the default database configuration
```bash
$ yarn setup
```

* Run the system
```bash
$ yarn dev
```

## Testing
Only run `yarn test` and all tests in the `__tests__/` diretory will be run.
